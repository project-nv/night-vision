# Range

Ranging indicator, e.g. RSI

## Data Format

```js
[<timestamp>, <value>]
```

## Range.color
- **Type:** `Color`
- **Default:** `'#ec206e'`

## Range.backColor
- **Type:** `Color`
- **Default:** `'#381e9c16'`

## Range.bandColor
- **Type:** `Color`
- **Default:** `'#535559'`

## Range.lineWidth
- **Type:** `number`
- **Default:** `1`

## Range.upperBand
- **Type:** `number`
- **Default:** `70`

## Range.lowerBand
- **Type:** `number`
- **Default:** `30`
